# PostABC 扩展安装说明

## 如何安装扩展

由于开发环境中的构建问题，您可以按照以下步骤直接加载扩展：

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并访问
3. 在页面右上角打开 "开发者模式" 开关
4. 点击 "加载已解压的扩展程序" 按钮
5. 选择项目目录 `D:\AiOnUiWorkspace\project\postabc`
6. 扩展将被加载到浏览器中

## 如何使用扩展

1. 安装完成后，您会在浏览器工具栏看到 PostABC 图标
2. 点击图标，会弹出一个窗口
3. 点击 "打开 API 调试界面" 按钮
4. 将在新标签页中打开 API 调试界面

## 功能

- API 调试界面：支持 GET, POST, PUT, DELETE 等 HTTP 方法
- 请求头、查询参数、请求体配置
- 环境变量管理
- 历史记录保存
- 支持 JSON、文本、表单数据等格式

## 注意事项

- 扩展需要网络权限和本地存储权限
- CORS 绕过功能可帮助调试跨域请求
- 所有数据都存储在本地浏览器中

现在您可以点击浏览器工具栏上的 PostABC 图标，然后点击弹出窗口中的 "打开 API 调试界面" 来开始使用 API 调试工具了！
